**job description**

Machine Operator

Machine Operators needed in the Athens area for a plastics manufacturer.

Pay Rate: $16.00/hr

YOU MUST:

- HAVE RECENT EXPERIENCE AS A MACHINE OPERATOR

- HAVE AT LEAST 6 MONTHS FORKLIFT EXPERIENCE IN THE PAST 18 MONTHS

- BE ABLE TO LIFT UP TO 50 POUNDS

- BE FLEXIBLE & WILLING TO WORK 12 HOUR SHIFTS, INCLUDING NIGHTS

Responsibilities:

You must be able to efficiently and safely perform the duties of a Machine Operator including set up, start up, troubleshooting and continuous operation of line equipment. See details below.

Set Up:

- Read and understand schedule, BOM, SOC.

- Be able to perform all functions of the set up including:

-- Select proper tooling

-- Install tooling

-- Insure correct resin

Start Up:

- Position all controls to SOC.

- Activate miscellaneous drives

- Test initial product

- Make necessary adjustments

- Verify product is in spec

Run Production:

- Package product in accordance with packaging instructions

- Troubleshoot the process

- Perform QC test per SOC

- Store finished product per instructions

- Validate Process is in Control

- Correctly Enters Data in Data Collection

Tooling:

- Identify Tooling damage and wear.

- Clean and return to stock all used tooling

Safety:

- Pinch Points Identification

- Hazard Material Communication

- Accident Reporting Procedure

- Water Hazards

- E-Stops Identification/awareness

- Completes necessary housekeeping duties in work area at all times

- Complies with all OSHA/Company safety requirements.

- Knowledgeable of location of MSDS, First Aid Supplies, fire extinguishers, exits.

Working hours: 8:00 PM - 8:00 AM
