Job Description
Job Description
A machinist will report directly to the area Group Leader. This position is responsible for the operation of machine tools of various types.
Responsibilities:
Essential Duties and Responsibilities
? Machine set up
o Assemble and install the tooling required per the set up instructions
o Install work holding devices
o Assemble in process inspection equipment required per the instructions
o Review work instructions for errors, omissions, and clarification that may be required
o Transfer any electronic data required to the machine
o Perform the first run of the operation
o Measure product and adjust machine as required
o Submit product to quality for first piece inspection and approval
o Inform the Manufacturing Engineer of any problems or concerns
o Train others on the operation as requested
? Machine operation
o Operate machinery per the instructions provided
o Measure product as it is produced to ensure conformity
o Accurately document activities as required
o Make adjustments to the machine as needed to maintain quality
o Change and adjust tooling as needed to ensure proper operation
o Maintain a clean and organized work area
o Accurately communicate conditions surrounding a failure of an operation
o Always maintain a high level of safety practices
? Machine set down
o Submit any work instruction corrections to the applicable Manufacturing Engineer
o Submit any electronic data from the machine tool to the applicable Manufacturing Engineer
o Remove all data and instructions from the work area
o Return inspection equipment to the appropriate place
o Remove and clean tooling and work holding devices. Return to the appropriate place
o Return unused, identified raw material to the material crib
