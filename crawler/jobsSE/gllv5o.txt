Job Title = Systems Engineer

Software Guidance & Assistance, Inc., (SGA), is searching for a Systems Engineer for a Contract assignment with one of our premier Financial Services clients in Raleigh, NC.

Responsibilities :
Day-to-day operational maintenance of Cyber Security agents and any associated backend infrastructure, including system administration tasks
Automating Windows-based tasks using Powershell
Writing code (in Powershell or Python) to interact with APIs of the deployed solutions
Document test plans and test results
Document solutions in wiki
Maintain relevant industry, information security, and technical knowledge
Required Skills:
5+ years of experience and excellent skills in the following:
Excellent Powershell development skills, including experience working with APIs and source code management
Excellent systems administration skills on Windows
Technical solution design, implementation, and integration experience in enterprise environments, preferably with information security software
Packaging and package integration on Windows
Decent working knowledge of Linux
