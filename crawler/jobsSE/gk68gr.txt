Job Title = DevOps Engineer

Software Technology Group DevOps Engineer
Summary
STG is looking for a DevOps Engineer who is passionate about automated provisioning, containers, and continuous deployment. This person needs to have the desire to improve, modernize and optimize software delivery utilizing the latest and greatest DevOps skills. This is a full-time salaried position with full benefits.
Job Responsibilities
Work on both the batch and the stream side of our Lambda Architecture.
Implement a continuous integration and deployment (CICD) pipeline.
Implement monitoring and alerting to increase uptime of our application.
Assist in the implementation of network based security.
Help develop our microservice strategy including overall design, technology decisions and implementation
Improve our software development process by modernizing our technology stack and leading other developers in architectural designs and implementations required for success
Conduct discussions, code reviews, design reviews and possible implementation demos
Interface with development and IT teams Required Skills & Experience
Experience with distributed applications
Have a solid understanding of software defined networks
Developing, debugging and deploying applications with Java, C#, or other similar languages
Linux admin with both OS and application orchestration and automation
Linux containers (Docker, Kubernetes, and Mesos)
Building and working with a Continuous Integration (CI) framework
Experience with Cloud platforms such as AWS, BlueMix, GAE, Azure, etc.
Experience with infrastructure automation using Cloud Formations, Terraform, Puppet, or Ansible
Experience with networking technologies, routers, switches, load balancing
Experience building automation tools and code coverage tools like Jenkins and Clover
KVM and/or VMWare virtualization
Experience in Agile/Scrum environment
Experience with Nutanix, OpenStack, or other hyper converged technologies is a plus Bonus Skills
Saltstack, Ansible, Puppet, Chef
Hadoop, DynamoDB
Docker, WeaveWorks
SMACK stack
Education Requirements
Bachelor of Science degree from an accredited university (Information Technology/Computer Science preferred) or equivalent experience
